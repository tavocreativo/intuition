== Description ==

Catálogo digital 2017 de perfumeria fina de Disaromas S.A, con los últimos productos de la alta perfumería. Este tema es desarrollado para Disaromas S.A. por Intuition Business.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in Disaromas in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to use your new theme right away.
4. Navigate to Appearance > Customize in your admin panel and customize to taste.
